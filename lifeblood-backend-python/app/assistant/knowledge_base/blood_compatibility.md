# Blood Group Compatibility

## Who can donate to whom (red blood cell compatibility)
- O- (O negative): can donate to all blood groups (universal donor). Can only receive from O-.
- O+ (O positive): can donate to O+, A+, B+, AB+. Can only receive from O+, O-.
- A-: can donate to A-, A+, AB-, AB+. Can receive from A-, O-.
- A+: can donate to A+, AB+. Can receive from A+, A-, O+, O-.
- B-: can donate to B-, B+, AB-, AB+. Can receive from B-, O-.
- B+: can donate to B+, AB+. Can receive from B+, B-, O+, O-.
- AB-: can donate to AB-, AB+. Can receive from AB-, A-, B-, O-.
- AB+ (AB positive): universal recipient, can receive from all blood groups. Can only donate to AB+.

## Quick summary
- Universal donor: O-
- Universal recipient: AB+
- Rh-negative blood groups (A-, B-, AB-, O-) can generally donate to both Rh-negative and Rh-positive recipients of compatible ABO type; Rh-positive blood groups can only donate to Rh-positive recipients.

## Why compatibility matters
Receiving an incompatible blood type can trigger a dangerous immune reaction. This is why hospitals and blood banks always perform a cross-match test before a transfusion, even if the donor and recipient blood groups are believed to be compatible.
